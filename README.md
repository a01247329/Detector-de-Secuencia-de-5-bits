Tema: Diseño y Producción de un Detector de Secuencia (Sequence Detector)

Detector de Secuencia de 5 bits
El sistema diseñado es un Detector de Secuencia de Moore que opera sobre una entrada serial. El sistema activa una salida 
cuando detecta la secuencia binaria 11010, utilizando la estructura de una maquina de estados finita.

Este repositorio incluye el modulo del programa, el banco de pruebas, diagrama de estados, y el análisis de producción industrial 
requeridos en la actividad.
